# HassGlass

Turn **Rokid AR Glasses** into a first-class voice + visual satellite for the Home Assistant **Assist** pipeline — no vendor cloud required.

- Hands-free voice control through your existing Assist pipeline.
- Glanceable HUD cards (weather, timers, doorbell, scene confirmations) on the micro-LED display.
- Touchpad / side-button / head-pose events surfaced as HA events.
- Per-glasses entities for battery, wearer presence, listening state, current card.

Inspired by [View Assist](https://github.com/dinki/view_assist_integration) — same mental model, swapped onto AR glasses.

See [README](https://github.com/teaganglenn/HassGlass) for setup.
